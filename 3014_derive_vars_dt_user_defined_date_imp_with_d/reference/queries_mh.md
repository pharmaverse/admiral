# Queries MH Dataset

Queries MH Dataset

## Usage

``` r
queries_mh
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 14
rows and 8 columns.

## Source

An example of standard query MH dataset to be used in deriving
Standardized MedDRA Query variables in ADMH

# Create a `derivation_slice` Object

Create a `derivation_slice` object as input for
[`slice_derivation()`](https:/pharmaverse.github.io/admiral/cran-release/copilot/enhance-examples-derive-basetype-records/reference/slice_derivation.md).

## Usage

``` r
derivation_slice(filter, args = NULL)
```

## Arguments

- filter:

  An unquoted condition for defining the observations of the slice

  Default value

  :   none

- args:

  Arguments of the derivation to be used for the slice

  A
  [`params()`](https:/pharmaverse.github.io/admiral/cran-release/copilot/enhance-examples-derive-basetype-records/reference/params.md)
  object is expected.

  Default value

  :   `NULL`

## Value

An object of class `derivation_slice`

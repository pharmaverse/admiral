# Single Dose Exposure Dataset (Removed)

**\[deprecated\]**

The `ex_single` dataset has been removed from `{admiral}` to reduce
package size. Derive it on the fly using
[`create_single_dose_dataset()`](https:/pharmaverse.github.io/admiral/copilot/2945-remove-ex-single-data/reference/create_single_dose_dataset.md).
See
[`vignette("occds", package = "admiral")`](https:/pharmaverse.github.io/admiral/copilot/2945-remove-ex-single-data/articles/occds.md)
for a full example.

## Usage

``` r
ex_single
```

## Format

An object of class `data.frame` with 0 rows and 0 columns.

# Queries Dataset

Queries Dataset

## Usage

``` r
queries
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 15
rows and 8 columns.

## Source

An example of standard query dataset to be used in deriving Standardized
MedDRA Query variables in ADAE

# List All Available ADaM Templates

List All Available ADaM Templates

## Usage

``` r
list_all_templates(package = "admiral")
```

## Arguments

- package:

  The R package in which to look for templates. By default `"admiral"`.

  Default value

  :   `"admiral"`

## Value

A `character` vector of all available templates

## See also

Utilities used for examples and template scripts:
[`use_ad_template()`](https:/pharmaverse.github.io/admiral/v1.4.1/reference/use_ad_template.md)

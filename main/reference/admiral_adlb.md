# Lab Analysis Dataset

An example of lab analysis dataset

## Usage

``` r
admiral_adlb
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 330
rows and 115 columns.

## Source

Derived from the `adlb` template, then further filtered due to dataset
size by the following USUBJIDs: 01-701-1015, 01-701-1023, 01-701-1028,
01-701-1033, 01-701-1034, 01-701-1047, 01-701-1097, 01-705-1186,
01-705-1292, 01-705-1310, 01-708-1286

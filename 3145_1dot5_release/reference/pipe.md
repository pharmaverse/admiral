# Pipe operator

See
`magrittr::`[`%>%`](https://magrittr.tidyverse.org/reference/pipe.html)
for more details.

## Usage

``` r
lhs %>% rhs
```

## Arguments

- lhs:

  A value or the magrittr placeholder.

  Default value

  :   `NULL`

- rhs:

  A function call using the magrittr semantics.

  Default value

  :   `NULL`
